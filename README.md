## Overview
HERO.IO is a task management app designed for teams. Users can create, assign, and track tasks efficiently with role-based access and a dashboard overview.  

## Tech Stack
- **Frontend:** React  
- **Backend:** Node.js, Express  
- **LocalStorage

##  Main Features
- Task CRUD (Create, Read, Update, Delete)  
- Role-based user authentication  
- Dashboard with team task overview  
- Responsive design for multiple devices  

##  Dependencies
- express  
- react-router-dom  
- axios
